SPACE GAME

I want to make a VR game where you get to watch a space battle unfold around you, watch ships tail and explode each other in glorious combat!



GOALS                                      
SHIPS
- ships should be able to:
 - chase an enemy ship
 - fire at enemy ship
 - on hit ship should explode
 - more ships should respawn to replace the destroyed ship
 - ships should move realistically as if they are using thrusters in a 0 gravity environment
 - ships should make a noise to indicate they have exploded
 - ships should be able to ask their allies for help if trailed - perhaps make a particle effect + sound to show this to user
 - if ship is trailed they should be able to dodge shots
 - ships should be distinguishable based on team
BACKGROUND
- large carrier ships where new ships spawn
- skybox should be distant stars
- should be planets to simulate a solar system



STRETCH GOALS                                                                             
- asteroid in the background to give the illusion of a dynamic universe
- reverb added to sound effects to simulate illusion of space
- trash ships used to collect debris of destroyed ships
- background planets should rotate and move
- player can in some way interact with the ships
